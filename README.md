## Run Project

Clone the project

```bash
  git clone repo.git
```
Go to the project directory

```bash
  cd anywall
```

### server
Go to the server directory

```bash
  cd server
```

Install dependencies

```bash
  yarn
```

Start dev the server (default PORT 8080)
**available PORT 3000, 5000, 8000, 8080**

```bash
  yarn dev
```

### client
Go to the server directory

```bash
  cd client
```

Install dependencies

```bash
  yarn
```

Start dev the server (default PORT 8080)
**available PORT 3000, 5000, 8000, 8080**

```bash
  yarn tauri dev
```
