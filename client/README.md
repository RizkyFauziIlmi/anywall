Install AnyWall with installer.

## Run Locally

Clone the project

```bash
  git clone https://github.com/RizkyFauziIlmi/anywall.git
```

Go to the project directory

```bash
  cd anywall
```

Install dependencies

```bash
  yarn
```

Start dev the server

```bash
  yarn tauri dev
```


## Deployment

To build this project run

```bash
  yarn tauri build
```
